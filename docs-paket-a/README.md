# DOCS PAKET A — WEBSITE ALFA BEAUTY (PROFESSIONAL B2B)

Paket A adalah website profesional B2B untuk positioning, katalog (tanpa harga publik), edukasi/event showcase, dan lead capture (Become Partner) dengan **lead terkirim ke email internal Perusahaan**.

Untuk mengurangi fragmentasi, seluruh spesifikasi Paket A kini digabung menjadi satu dokumen canonical:

- **Baca/ubah di sini:** `docs-paket-a/paket-a.md`

Proposal komersial (Owner-ready):

- **Baca/ubah di sini:** `docs-paket-a/proposal.md`

Runbook triage (operasional: gejala → pengecekan cepat → tindakan):

- `docs-paket-a/runbook-triage.md`

Dokumen-dokumen lama yang sudah disupersede disimpan sebagai arsip di:

- `docs-paket-a/_deprecated/`

Catatan: isi folder `_deprecated/` adalah arsip pendekatan lama (mis. persistence/export/RUM) dan **bukan** acuan untuk scope final Paket A.

> Paket A sengaja **tidak** mencakup modul platform transaksi (login multi-role, tier pricing, credit, ERP sync, SLA routing, invoice, loyalty, dsb). Itu ada di `docs-paket-b/`.
