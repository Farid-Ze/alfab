# Security Policy

## Supported Versions

| Version | Supported |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x: |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability:

1.  **Do NOT create a public issue.**
2.  Email our security team at `security@alfabeautycosmetica.com`.
3.  Include a description of the vulnerability and steps to reproduce.

## Response Timeline

- **Acknowledgement:** 24 hours
- **Assessment:** 3-5 business days
- **Patches:** Released as `hotfix` or priority release.

## Scope

- Website Infrastructure (Next.js, Vercel)
- Lead Capture API
- Database Integrity
